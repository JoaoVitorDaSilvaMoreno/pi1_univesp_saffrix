# 🚜 Saffrix Sistema de Gerenciamento de Frotas Agrícolas - Projeto Integrado 1 Univesp

## 🎯 Objetivo
Este sistema visa o gerenciamento eficiente de manutenções preventivas para frotas agrícolas, garantindo que os veículos estejam sempre em condições ideais para operação no campo.

---

## ⚙️ Como funciona o sistema

- Quando um veículo sai para o campo, o operador informa no sistema o odômetro antigo para registrar o uso até o retorno.
- O sistema monitora o tempo de uso e dispara alertas automáticos:
  - ⚠️ Alerta de manutenção próxima quando faltam 20 horas para o vencimento da manutenção.
  - ❌ Indicação de manutenção vencida quando o prazo é ultrapassado.
- O sistema registra o histórico de uso e as manutenções realizadas para cada veículo.

---

## 🛠️ Funcionalidades principais

- 📋 Cadastro de veículos com informações detalhadas:
  - Nome do veículo  
  - Placa  
  - Número de série  
  - Odômetro  
  - Tempo para realização das manutenções  

- 📊 Dashboard com indicadores:
  - Manutenções realizadas  
  - Quantidade de veículos com manutenção vencida, próxima ou em dia  
  - Veículos inativos com histórico de inatividade  

- 🔐 Controle de acesso com dois níveis:
  - Usuário  
  - Administrador  

- 🕒 Histórico completo de uso e manutenções realizadas.

---

## 👥 Perfis de acesso

- **Usuário:** Pode registrar saídas e retornos dos veículos, visualizar alertas e consultar histórico.  
- **Administrador:** Além das funções do usuário, pode cadastrar veículos, gerenciar usuários e acessar dashboards completos.

---

## ▶️ Como usar

1. Faça o login com seu usuário e senha.  
2. Cadastre os veículos e configure os parâmetros de manutenção.  
3. Registre as saídas e retornos dos veículos para monitorar o odômetro.  
4. Acompanhe os alertas de manutenção e realize as manutenções preventivas.  
5. Utilize os dashboards para análise e tomada de decisão.
